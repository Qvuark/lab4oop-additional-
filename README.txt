Опис роботи програми на базі бібліотеки SFML

ВСТУП:
Ця програма демонструє створення, відображення та рух фігур на екрані з використанням бібліотеки SFML. SFML (Simple and Fast Multimedia Library) — це кросплатформна бібліотека, яка використовується для розробки графічних застосунків, зокрема ігор.

ЗАГАЛЬНА ЛОГІКА ПРОГРАМИ:
Програма створює вікно розміром 800x600 пікселів із заголовком "Dynamic Figures". У цьому вікні відображаються кола, квадрати та ромби, які користувач може додавати, натискаючи відповідні клавіші:
- C — додає коло.
- S — додає квадрат.
- R — додає ромб.

Усі створені фігури автоматично рухаються вправо на фіксовану відстань у кожному кадрі(50 мс затримки базово).

СТРУКТУРА ПРОГРАМИ:
1. Класи фігур:
   - Circle (Коло)
   - Square (Квадрат)
   - Rhomb (Ромб)

Кожен клас описує поведінку та відображення своєї фігури. Ці класи містять:
- Поля для координат та параметрів фігури (наприклад, радіус кола або довжини діагоналей ромба).
- Конструктор, який задає початкові значення.
- Метод `Draw()`, який відображає фігуру на екрані.
- Метод `MoveRight()`, який зміщує фігуру вправо.
- Метод `UpdatePosition()`, який оновлює положення фігури на екрані.

2. Основний цикл програми:
   1. Відкривається вікно застосунку.
   2. Користувач може додавати фігури за допомогою клавіатури.
   3. У кожній ітерації виконуються наступні кроки:
     - Перевірка користувацьких подій (закриття вікна, натискання клавіш).
     - Очищення екрана білим кольором.
     - Оновлення положення всіх фігур та їх відображення.
     - Оновлення вмісту вікна (`window.display()`).
     - Затримка для плавного відображення.

ЯК ПРАЦЮЄ SFML:
SFML — це потужна бібліотека, яка надає класи для роботи з графікою, звуком та введенням. У нашій програмі ми використовуємо її для роботи з вікнами (`sf::RenderWindow`), подіями (`sf::Event`) та геометричними примітивами (`sf::CircleShape`, `sf::RectangleShape`, `sf::ConvexShape`).

Ось основні етапи роботи SFML у програмі:
- Створення вікна із заданими параметрами.
- Обробка подій через цикл `while (window.pollEvent(event))`.
- Відображення фігур за допомогою методів `window.draw()`.
- Оновлення екрана через `window.display()`.

ВИСНОВОК:
Ця програма демонструє основи роботи з графічними елементами та рухом на екрані з використанням SFML. Ми використали об’єктно-орієнтований підхід, створивши класи для кожної фігури. Це спрощує розширення програми — наприклад, можна додати нові фігури або змінити логіку руху, модифікувавши відповідні класи.

Виконання цього проекту допомагає зрозуміти такі концепції, як: керування подіями, цикли оновлення графіки та використання ООП у графічних застосунках. Таким чином, SFML — чудова бібліотека для створення графічних проєктів мовою C++.
